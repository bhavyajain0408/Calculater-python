# Simple Calculator in Python 🧮

A beginner-friendly Python calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, division, and average calculation through a command-line interface.

## Features

* Addition of two numbers
* Subtraction operation
* Multiplication calculation
* Division functionality
* Average calculation
* Interactive user input system
* Simple and clean Python implementation

## Technologies Used

* Python Programming Language

## Project Overview

This project demonstrates the use of Python functions, conditional statements, and user input handling to build a functional calculator. It is designed to strengthen core programming concepts and improve logical problem-solving skills for beginners.

## Learning Outcomes

Through this project, I practiced:

* Function creation and reuse
* Conditional logic (`if-elif-else`)
* User input handling
* Mathematical operations in Python
* Building interactive console applications
